## Шаблоны для квартир, поставщиков и услуг

Сделаем все базовые шаблоны для квартир, поставщиков и услуг.

Отлично! Давай сразу сделаем **базовые Blade-шаблоны для CRUD** по трём сущностям: квартиры, поставщики и услуги. Они будут работать с минимальной функциональностью: просмотр списка, создание, редактирование и просмотр деталей.

---

## **1. Квартиры (Apartments)**

### `resources/views/apartments/index.blade.php`

```blade
@extends('layouts.app')

@section('content')
<h1>Квартиры</h1>
<a class="btn btn-primary mb-3" href="{{ route('apartments.create') }}">Добавить квартиру</a>
<table class="table table-bordered">
    <thead>
        <tr>
            <th>ID</th>
            <th>Название</th>
            <th>Адрес</th>
            <th>Действия</th>
        </tr>
    </thead>
    <tbody>
        @foreach($apartments as $apartment)
        <tr>
            <td>{{ $apartment->id }}</td>
            <td>{{ $apartment->name }}</td>
            <td>{{ $apartment->address }}</td>
            <td>
                <a class="btn btn-sm btn-info" href="{{ route('apartments.show', $apartment) }}">Просмотр</a>
                <a class="btn btn-sm btn-warning" href="{{ route('apartments.edit', $apartment) }}">Редактировать</a>
                <form action="{{ route('apartments.destroy', $apartment) }}" method="POST" style="display:inline;">
                    @csrf
                    @method('DELETE')
                    <button class="btn btn-sm btn-danger" type="submit">Удалить</button>
                </form>
            </td>
        </tr>
        @endforeach
    </tbody>
</table>
@endsection
```

### `resources/views/apartments/create.blade.php`

```blade
@extends('layouts.app')

@section('content')
<h1>Добавить квартиру</h1>
<form action="{{ route('apartments.store') }}" method="POST">
    @csrf
    <div class="mb-3">
        <label>Название</label>
        <input type="text" name="name" class="form-control" required>
    </div>
    <div class="mb-3">
        <label>Адрес</label>
        <input type="text" name="address" class="form-control">
    </div>
    <button class="btn btn-success" type="submit">Сохранить</button>
</form>
@endsection
```

### `resources/views/apartments/edit.blade.php`

```blade
@extends('layouts.app')

@section('content')
<h1>Редактировать квартиру</h1>
<form action="{{ route('apartments.update', $apartment) }}" method="POST">
    @csrf
    @method('PUT')
    <div class="mb-3">
        <label>Название</label>
        <input type="text" name="name" class="form-control" value="{{ $apartment->name }}" required>
    </div>
    <div class="mb-3">
        <label>Адрес</label>
        <input type="text" name="address" class="form-control" value="{{ $apartment->address }}">
    </div>
    <button class="btn btn-success" type="submit">Сохранить</button>
</form>
@endsection
```

### `resources/views/apartments/show.blade.php`

```blade
@extends('layouts.app')

@section('content')
<h1>Квартира: {{ $apartment->name }}</h1>
<p><strong>Адрес:</strong> {{ $apartment->address }}</p>
<a class="btn btn-primary" href="{{ route('apartments.index') }}">Назад</a>
@endsection
```
